## 1.1 일반적인 설계 문제를 특정한 맥락으로 해결하기 위해 객체와 클래스를 전달하는 방법을 맞춘 것 - GoF

1. creational: 객체 생성, 초기화, 클래스 선택에 관한 패턴 (싱글톤, 팩토리)
2. behavior: 객체 간 소통, 메시징, 상호 작용에 관한 패턴  (옵저버)
3. structural: 클래스와 객체 관계를 조직하는 패턴, 관련된 객체를 서로 묶어 원하는 조직을 구현하는 방법 안내 (장식자)
